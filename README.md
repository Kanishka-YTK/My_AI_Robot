# 🤖 AI-Based Autonomous Navigation System

## 🌟 Project Overview
This is a 2D simulation of an autonomous robot. It uses Artificial Intelligence to find the shortest path between two points while avoiding obstacles in real-time.

## 🚀 How It Works
1. **Perception:** The system "sees" the grid and identifies obstacles (Black blocks).
2. **Path Planning:** Uses the **A* Algorithm** to calculate the best route.
3. **Decision Making:** The AI decides which step to take next to reach the goal (Red block).

## 🛠 Tech Stack
* **Language:** Python 3.14.4
* **Library:** Pygame
* **Algorithm:** A* Search

## 🎮 How to Run
1. Install Python and Pygame (`pip install pygame`).
2. Run `python src/main.py`.
3. **Controls:**
   - Left Click: Place Start (Green) / End (Red) / Walls (Black).
   - Spacebar: Start the AI.
   - 'C' Key: Clear the screen.
